VS Code Extension for Web Development

Visual Studio Code is a powerful code editor made by Microsoft, with its amazing support for all programming languages, it has become the favourite of programmers, especially in the web development community it has gained more acknowledgement because of its inbuilt support for JavaScript, TypeScript nad Node.js.

There are many extensions for all programming languages in VS code, I am going to put the extension which you can use on a daily basis for web development.

Bracket Pair Colorizer

Bracket Pair Colorizer allows matching brackets to be identified with the same colours, by default (), [] and {} are matched with the same colour.When brackets are nested together it becomes visually difficult to identify the opening and the closing brackets. Bracket Pair Colorizer becomes useful at such times where it adds colours to bracket pairs so even if there is a nested set of brackets, it's easy to visually identify the pair of brackets.

After installing this extension you can press Shift+Alt+Right and the extension will do what it does the best.

Prettier - Code Formatter

Every programmer desires to have a beautiful looking code but most of them are not able to format it beautifully.Prettier code formatted takes care of that by formatting the code in its own way, like removing unnecessary spaces between the lines, taking into account word wrap and other such factors.

After installing this extension you can press Ctrl/Cmd+Shift+P which will open a command palette and then type Format Document or you can select a section then open the command palette and type Format Selection

Live Server

Running an HTML to visit the webpage you are creating has never been easy, finding the HTML file then clicking on it to run on the browser, too much effort.Live server extension solves this by adding a local development server with live load and reload features.

After installing this extension you have to click on GoLive which at the bottom right of your VScode editor, or you can right-click on your editor and click on Open Live Server Option and you can see your HTML page in a browser.

HTML CSS Support

HTML CSS Support extension helps you find a list of id and class attributes suggestions.After installing the extension press Ctrl/Cmd + Space inside a tag and it will show you a list of attributes that you can add to your element.
